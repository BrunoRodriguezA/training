Herencia 
	clase padre - superclase 
		Desde la clase padre no podemos acceder a métodos de la clase hija 
	clase hijas - subclases 
		Desde la clase hijo si podemos acceder a métodos de la clase padre

	Sobreescritura - clase hija el método existente de la clase padre 
		el método ha de tener la misma firma (mismo nombre) utilizado en la clase padre, un estándar de Python 

Polimorfismo - Multiples comportamientos en distintas clases, según el tipo de dato con el que estemos trabajando y si existe o no sobreescritura

	gerarquias entre clases 
	interface - uso del mismo nombre del método, comportamiento entre varias clases 


	Función polimórfica - duck typing : el tipo de objeto se determina según su comportamiento
		si parace pato y se comporta como un pato, entonces es un pato 

Clase object - clase padre de todas las clases en Python de manera directa/indirecta
